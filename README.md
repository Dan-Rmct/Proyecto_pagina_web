# Proyecto-Pagina-Web

Diseño de la página web oficial del Club Hello World de la FES Aragón, UNAM,
desarrollada para la primera edición del Hackathon 2026.

## Integrantes
- Aldo Daniel Ramírez Cortés
- Valeria Vianey Rodríguez
- Martínez Figueroa Erick

## Tecnologías Utilizadas
- **Framework:** Astro v5
- **Estilos:** Tailwind CSS v4
- **Lenguaje:** TypeScript
- **Gestor de paquetes:** pnpm
- **Despliegue:** GitHub Pages
- **Diseño:** Figma

## Prerequisitos
- Node.js v18 o superior
- pnpm (`npm install -g pnpm`)

## Instrucciones de Ejecución
1. Clona el repositorio:
   `git clone https://github.com/Dan-Rmct/Proyecto_pagina_web.git`
2. Instala las dependencias:
   `pnpm install`
3. Inicia el servidor de desarrollo:
   `pnpm dev`
4. Abre tu navegador en `http://localhost:4321`

## Estructura del Proyecto
```
/
├── public/          # Imágenes y assets estáticos
├── src/
│   ├── components/  # Componentes reutilizables (Header, CardEquipo, etc.)
│   ├── layouts/     # Layout base de la página
│   ├── pages/       # Páginas del sitio
│   └── styles/      # Estilos globales
├── docs/            # Documentación del proyecto
└── README.md
```
